## Google Sheets setup

The appointment form posts to `src/app/api/saveToGoogleSheet/route.js`, which requires server-side environment variables in the project root `.env` file.

1. Copy `.env.example` to `.env`.
2. Fill in `GOOGLE_CLIENT_EMAIL` and `GOOGLE_PRIVATE_KEY` with your Google service account credentials.
3. Keep the spreadsheet IDs as-is unless you want to write to different sheets.
4. Share both Google Sheets with the service account email so it can append rows.
5. Restart `npm run dev` after changing `.env`.

Do not use `NEXT_PUBLIC_` for these secrets. They must stay server-side.
